# Scanner de Portas

Ferramenta de seguranca de rede para varredura de portas e deteccao de servicos e vulnerabilidades.

## Funcionalidades
- Varredura de portas com Nmap
- Deteccao de vulnerabilidades com Nikto
- Relatorio automatico de servicos expostos

## Tecnologias
- Python 3
- Nmap
- Nikto

## Uso
`ash
pip install -r requirements.txt
python aut_vul.py
``n
